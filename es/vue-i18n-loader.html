# vue-i18n-loader

Si desea habilitar vue-i18n-loader, simplemente configure la opción vueI18nLoader en true.

// nuxt.config.js

['nuxt-i18n', {
  vueI18nLoader: true
}]

Ahora puede definir traducciones utilizando bloques personalizados en sus archivos Vue:

<i18n>
{
  "en": {
    "hello": "hello world!"
  },
  "ja": {
    "hello": "こんにちは、世界!"
  }
}
</i18n>

<template>
  <p>{{ $t('hello') }}</p>
</template>

# YAML

<i18n lang="yaml">
en:
  hello: "hello world!"
ja:
  hello: "こんにちは、世界!"
</i18n>

<template>
 <p>{{ $t('hello') }}</p>
</template>
